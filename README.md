# Система учёта книг в библиотеке

## Описание
Проект представляет собой информационную систему для **автоматизации учёта книг в библиотеке**.  
Система обеспечивает хранение, выдачу, возврат и отслеживание литературы, а также взаимодействие между библиотекарями и читателями.

## Цели
- Сократить количество ошибок при ведении учёта.  
- Ускорить процесс выдачи и возврата книг.  
- Обеспечить удобный поиск литературы по различным параметрам.  
- Вести историю всех операций с книгами.  
- Предоставить читателям возможность просматривать доступные книги и свои активные выдачи.

## Пользователи системы
- **Библиотекарь** — администратор, управляющий каталогом и учётом выдачи книг.  
- **Читатель** — зарегистрированный пользователь, который просматривает каталог и отслеживает свои выдачи.

## Основные функции
- Добавление, редактирование и удаление книг.  
- Регистрация и управление читателями.  
- Выдача и возврат книг.  
- Просмотр каталога доступных книг.  
- Формирование отчёта о просроченных книгах.

## Технологии
- **Язык программирования:** Python  
- **Фреймворк:** Flask  
- **База данных:** SQLite  
- **Архитектура:** клиент-серверная
